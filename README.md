# 🛒 Ecommerce – Full Stack Next.js

Aplicación **Ecommerce completa** desarrollada con Next.js, enfocada en resolver
problemas reales de una tienda online moderna: autenticación, carrito, órdenes,
panel de administración y render dinámico desde servidor.

## 🚀 Tecnologías

- Next.js 15 (App Router)
- TypeScript
- MongoDB Atlas + Mongoose
- NextAuth.js
- Tailwind CSS
- Zustand
- Vercel

## ✨ Funcionalidades principales

- Registro y login de usuarios
- Recuperación y cambio de contraseña
- Carrito de compras persistente
- Gestión de órdenes
- Panel de administración (productos, categorías, pedidos)
- Renderizado dinámico desde API
- Manejo de estados de carga y errores
- Protección de rutas con middleware
- Manejo avanzado de imágenes con `next/image`

## 🧠 Arquitectura

- Backend integrado con App Router (`/app/api`)
- Modelos desacoplados (Mongoose)
- Providers y contextos separados
- Store global con Zustand
- Middleware para control de sesión y permisos
